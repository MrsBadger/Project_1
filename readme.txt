Проект состоит из двух файлов: bot.py и kernel.py
В kernel.py прописана логика работы с элементами страницы для создания листа, состоящего из часов отправления ещё не ушедших на момент запуска бота электричек.
В bot.py прописана основная логика работы с ботом. 
Запуск происходит через запуск скрипта bot.py.
Работа с ботом начинается с команды /start.
